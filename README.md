# 🎵 Загрузчик Яндекс.Музыки

Удобная программа для скачивания вашей медиатеки из Яндекс.Музыки в **MP3** или **FLAC/Lossless** и экспорта списка треков в **CSV** или **JSON**.

Поддерживает:
- авторизацию через код подтверждения;
- интерактивное меню на русском языке;
- экспорт списка треков без лишних технических полей;
- скачивание в максимально доступном качестве (`MP3` max bitrate или `FLAC/Lossless`, если доступно);
- **вшивание тегов прямо в файл** (название, исполнитель, альбом, год, номер трека/диска, жанр, текст, обложка);
- сохранение музыки по папкам `Исполнитель/` (по умолчанию, без альбомов);
- выбор, скачивать обложки альбомов или нет;
- возврат в главное меню после выполнения действия.

---

## Краткая инструкция

1. Установите Python 3.
2. Запустите `start.bat`.
3. Программа сама проверит зависимости и откроет главное меню.
4. При первом запуске выберите `1` и пройдите авторизацию через код.
5. После этого можно:
   - экспортировать список треков в `CSV` или `JSON`;
   - скачать всю библиотеку в `MP3`;
   - скачать всю библиотеку в `FLAC/Lossless`;
   - выбрать, сохранять ли обложки альбомов.

---

## Что создаёт программа

- `token.txt` — сохранённый токен авторизации
- `library.csv` — экспорт списка треков
- `library.json` — экспорт списка треков
- папку с музыкой, где треки раскладываются по структуре:

```text
Яндекс.Музыка/
├── Исполнитель/
│   ├── Исполнитель - Название трека.mp3
│   └── cover - Альбом.jpg
```

---

## Формат экспорта

В экспортированных файлах остаются только полезные поля:
- `Название`
- `Исполнитель`
- `Альбом`
- `Год`
- `Длительность`

CSV сохраняется в формате, удобном для открытия в Excel.

---

## Запуск без меню

Если нужно, программу можно запускать и напрямую:

```cmd
python downloader.py --auth
python downloader.py --list --export-format csv
python downloader.py --download --quality mp3 --with-covers
python downloader.py --download --quality lossless --concurrency 4
python downloader.py --download --quality lossless --album-folders
```

---

## Важно

- FLAC/Lossless доступен не для всех треков и зависит от подписки.
- Уже скачанные файлы не загружаются повторно, если не отключать пропуск вручную.
- Если обложка доступна, программа может сохранить её как `cover.jpg` в папке альбома.
- Если скачивание трека упало, выполняется **вторая попытка** автоматически.
